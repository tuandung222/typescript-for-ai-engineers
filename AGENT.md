# AGENT.md

## Project overview

This repository is a Vietnamese Docusaurus curriculum book named TypeScript for AI Engineers. The audience is data scientists, research scientists, AI engineers, AI solution architects and software architects who want to learn TypeScript through production code.

## Content standard

Write in professional Vietnamese lecturer style. Build intuition first, then formalize. Use English technical terms when they are standard. Avoid jokes, slang and overly casual wording.

## Source policy

Docmost is used as an architecture case study. Do not copy long prose from its repository. Summarize and explain in original Vietnamese. Code snippets should be short and used only when they clarify a concept.

## Privacy and repository constraints

README.md must remain empty. The site must keep robots noindex,nofollow,noarchive,nosnippet, robots.txt must disallow all crawling and sitemap must stay disabled.

## Commands

Use npm run qa:docs for documentation QA. Use npm run verify before pushing. Do not mutate dependencies unless the task requires it.

## Completion checklist

Before finishing, check README.md size, run npm run verify, confirm sidebar links exist, push to GitHub and verify GitHub Pages deployment if deployment is part of the task.
